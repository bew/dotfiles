My personnal zsh config

![Bew's zsh config](media/zsh.png)

## Installation

### clone

```shell
cd ~/.config && git clone --recursive git@github.com:bew/configzsh.git zsh
```

### place config files

```shell
ln -s ~/.config/zsh ~/.zsh
ln -s ~/.zsh/zshenv ~/.zshenv
ln -s ~/.zsh/zshrc ~/.zshrc
ln -s ~/.zsh/zlogin ~/.zlogin
```
