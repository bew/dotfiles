Talk directly to the user you are assisting.

This command is usually used when the automated thinking and experimentation loop of the model is
stuck trying to resolve a seemingly insignificant problem, or doesn't progress in the right direction.
The user might be able to help you go in the right direction, or even solve the current issue for you instead so that experimentation / thinking can continue on other tasks.

Think about the current session and recent actions:

- Briefly summarize the current goal in 1–2 sentences.
- List any constraints/unknowns/blockers you recently hit and are trying to resolve.
- Ask the user a few concrete questions whose answer will help you unblock the current issues or avoid repeated brute-force attempts.

Focus on being clear and specific so the user can quickly unblock you.
